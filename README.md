# Data Analytics End-to-End Project

## Overview
This project demonstrates an end-to-end data analytics workflow. It covers data ingestion, exploration, cleaning, SQL analysis, and dashboarding to deliver actionable insights. The goal is to showcase practical skills in Python, SQL, and Power BI for real-world data problems.

## Dataset
**Source**: https://github.com/chichiracheal/customer_shopping_analysis/blob/main/customer_shopping_behavior.csv  
**Description**: Brief description of what the dataset contains, number of rows/columns, and key fields used in the analysis.

## Tools
- **Python**: Pandas, NumPy, Matplotlib, Seaborn for data loading, EDA, and cleaning
- **PostgreSQL**: SQL queries for data extraction and aggregation
- **Power BI**: Interactive dashboard for visualization and storytelling
- **Gamma**: Presentation deck for stakeholder reporting
- **GitHub**: Version control and project hosting

## Steps
1. **Data Loading**: Import raw dataset into Python using Pandas.
2. **Exploratory Data Analysis**: Identify distributions, missing values, outliers, and correlations.
3. **Data Cleaning**: Handle missing values, remove duplicates, standardize formats, and create derived features.
4. **SQL Analysis**: Load cleaned data into PostgreSQL and run queries to answer key business questions.
5. **Dashboard Development**: Build an interactive Power BI dashboard with filters, KPIs, and visuals.
6. **Reporting**: Summarize findings in a written report and create a slide deck using Gamma.

## Dashboard Results
- **Key Insight 1**: % of Customers by Subscription Status
- **Key Insight 2**: Revenue by Category  
- **Key Insight 3**: Revenue by Age Group

## How to Run It
1. Clone this repository: `(https://github.com/chichiracheal/customer_shopping_analysis)`
2. Install dependencies: `pip install -r requirements.txt`
3. Run `(https://github.com/chichiracheal/customer_shopping_analysis/blob/main/customer_behavior.ipynb)` to reproduce the EDA and cleaning steps.
4. Execute SQL scripts in `https://github.com/chichiracheal/customer_shopping_analysis/blob/main/customer_behavior.sql` on your PostgreSQL instance.
5. Open the `https://github.com/chichiracheal/customer_shopping_analysis/blob/main/customer_behavior.pbix` file in Power BI to view the dashboard.
